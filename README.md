## 算法的学习方式

| 区别 |     监督学习      | 无监督学习 |
| :--: | :---------------: | :--------: |
| 特征 |         ✅️         |     ✅️      |
| 标签 |         ✅️         |     ❌️      |
| 任务 | 分类任务/回归任务 |  聚类任务  |

### 有监督学习

| 算法任务 | 标签类型 |   案例   |
| :------: | :------: | :------: |
| 分类算法 |   离散   | 垃圾邮件 |
| 回归算法 |   连续   | 房价预测 |
